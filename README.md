# Micro Grand Maison Web- Frontend Web Application

Micro Grand Maison の可視化ダッシュボードおよびチャットクライアントを提供するフロントエンド・アプリケーション。

## 🌟 主な機能

* **2D店舗マップ風ダイアグラムのレンダリング**:
  * `ReactFlow` を用いて、擬人化されたマイクロサービスアバター（レストランスタッフ）とサービス間の通信パス（Edge）をインタラクティブなマップ上に表示します。
  * 選択されたノードと接続されている Edge のみ通信プロトコル名（HTTP/gRPC/PubSub 等）を動的にラベル表示する、ホバー時ラベル強調機能を搭載。
  * マイクロサービス間のネットワーク通信方向を示すアニメーション Edge (`strokeDasharray`, `markerEnd` 矢印)。
* **キャラクター・チャットドロワー**:
  * 画面右側のスライド式ドロワーに、キャラクターの詳細プロフィール、使用技術スタック、接続されているインカム/アウトカム依存サービス一覧を表示します。
  * スタッフと直接チャットで会話ができる対話画面（スレッド形式）を配置。ゲストユーザー用のメモリベースチャットおよびログインユーザー用の永続化チャットをサポート。
* **リアルタイム進捗と同期制御**:
  * プロジェクトの作成時や手動アップデート時、バックエンドから返る進捗フェーズ（`current_phase` 例：「Cloning repository...」「Generating avatars (3/6)...」など）をリアルタイムでポーリングし、進捗バーと点滅インジケーターで滑らかに表示します。
* **スマート通知システム**:
  * 画面右上のベルマークメニューから、プロジェクト解析結果や Git push Webhook の受信履歴をタイムライン形式で確認できます。
  * 各通知カードの `×` ボタンをクリックして個別にタイムラインから非表示（ローカルブラックリスト登録）にでき、その設定は `localStorage` に保存されます。

## 📂 主要ディレクトリ・コード構造

```
src/
├── app/
│   ├── page.tsx           # メインダッシュボード（プロジェクト作成、リポジトリ登録、プロジェクト/サンプル一覧）
│   ├── project/[id]/      # 2D ダイアグラムマップ表示、ReactFlow、チャットドロワー、手動アップデート
│   └── layout.tsx         # ルートレイアウト、Firebase プロバイダー
├── components/
│   └── Header.tsx         # 共通ヘッダー、通知ベル（Web/Git通知）、ログイン・ログアウト管理
└── lib/
    └── firebase.ts        # Firebase Authentication 設定（GitHub OAuth 認証）
```
